# О тестовом задании

Страницы интернет-магазина по продаже аудио аксессуаров. Главная страница - каталог товаров, вторая - корзина с приобретенными товарами. Реализовано полное взаимодействие с товарами.

## Быстрый старт

1. Установить зависимости `npm i`
2. Запустить проект в режиме разработки `npm run dev`

## Стек Технологий

- **Frontend**: React 19, TypeScript
- **Хранение**: SessionStorage
- **Роутинг**: React Router
- **Стили**: SCSS Modules
- **Бандлер**: Vite
